# GalleryShoppingMall
> Vue.js와 SpringBoot를 활용한 프로젝트   


<br>

## 1.참여 인원
+ 개인 프로젝트

<br>

## 2. 사용기술

#### `Back-end`
+ Java 9
+ SpringBoot 3.2.1
+ Gradle 7.6.1
+ JPA 
+ MariaDB

#### `Front-end`
+ Vue.js
+ Bootstrap 템플릿


<br>
